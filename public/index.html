En server för test av postdata

Denna server tar emot HTTP requests med metoden POST och returnerar definierade parametrar i HTTP HEADER och BODY. Parametrar som returneras är h_temp, h_humid och b_temp, b_humid.

  • IP
  • method: POST
  • Content-Type: application/x-www-form-urlencoded
  • Port: 3000
  • param1: h_temp (data i header)
  • param2: h_humid (data i header)
  • param3: b_temp (data i body)
  • param4: b_humid (data i body)

Exempel på REQ

http://<ip>:3000/api

ESP8266

"h_temp=" + String(temp) + "&h_humid=" + String(humid)

Postman

Content-Type: application/x-www-form-urlencoded

Data i HEADERS eller BODY
